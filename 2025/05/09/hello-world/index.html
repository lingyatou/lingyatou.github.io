基础用法说明

Front-matter

Front-matter 是 markdown 文件最上方以 --- 分隔的区域，用于指定个别档案的变数。

  • Page Front-matter 用于页面配置
  • Post Front-matter 用于文章页配置

如果标注为可选的参数，可以根据需要添加，不必全部填写。

Page Front-matter：

写法 解释
title 【必需】页面标题
date 【必需】页面创建日期
type 【必需】仅标籤、分类和友情链接页面需要配置
updated 【可选】页面更新日期
description 【可选】页面描述
keywords 【可选】页面关键字
comments 【可选】是否显示评论模块（默认 true）
top_img 【可选】页面顶部图片
mathjax 【可选】是否启用 MathJax（需设定 mathjax.per_page: false 才需配置）
katex 【可选】是否启用 KaTeX（同上）
aside 【可选】是否显示侧边栏（默认 true）
aplayer 【可选】是否加载 APlayer 的 js 和 css
highlight_shrink 【可选】代码框是否折叠（true/false，默认跟随配置）

Post Front-matter：

写法 解释
title 【必需】文章标题
date 【必需】文章创建日期
updated 【可选】文章更新日期
tags 【可选】文章标签
categories 【可选】文章分类
keywords 【可选】文章关键字
description 【可选】文章描述
top_img 【可选】文章顶部图片
cover 【可选】文章缩略图（优先级低于 top_img，可设为 false/图片地址/留空）
comments 【可选】是否显示评论模块（默认 true）
toc 【可选】是否显示文章目录（默认根据配置）
toc_number 【可选】是否显示 TOC 编号（默认根据配置）
toc_style_simple 【可选】是否使用简洁目录模式
copyright 【可选】是否显示版权模块（默认根据配置）
copyright_author 【可选】文章作者
copyright_author_href 【可选】作者链接
copyright_url 【可选】文章链接
copyright_info 【可选】版权声明
mathjax 【可选】是否启用 MathJax（需设定 mathjax.per_page: false 才需配置）
katex 【可选】是否启用 KaTeX（同上）
aplayer 【可选】是否加载 APlayer 的 js 和 css
highlight_shrink 【可选】代码框是否折叠（true/false，默认跟随配置）
aside 【可选】是否显示侧边栏（默认 true）
