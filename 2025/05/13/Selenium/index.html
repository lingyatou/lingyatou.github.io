什么是Selenium

Selenium 是一个功能强大的自动化测试工具，主要用于对 Web 应用程序进行功能测试和回归测试。它支持多种编程语言（如 Python、Java、C# 等）以及主流浏览器（Chrome、Firefox、Edge 等）。通过模拟用户在浏览器中的操作，Selenium 可以帮助开发者快速验证应用程序的功能是否正常。

WebDriver

WebDriver 是与浏览器交互的核心组件。每个浏览器都有其对应的WebDriver
如谷歌浏览器的ChromeDriver
你可以使用系统环境变量

也可以直接指定WebDriver文件位置

核心功能

WebElement

WebElement 是 Selenium 中表示页面元素的接口。通过定位元素（如输入框、按钮等），可以对它们进行操作。
以下是各定位类

By.id() - 通过 HTML id 属性

适用场景：元素有唯一id

By.name() - 根据表单字段名称定位

适用场景：表单元素（如 input name=”q”>）

By.className() - 通过 class 属性

⚠️ 注意：如果 class 有多个值（如 class=”btn btn-primary”），必须匹配完整：

By.tagName() - 通过 HTML 标签名

适用场景：批量获取同类标签元素（如所有input>或 a>）

By.linkText() - 通过超链接的完整文本

适用场景：精确匹配 标签的文本（如 a href=”#”>登录 /a）

By.partialLinkText() - 通过超链接的部分文本

适用场景：模糊匹配链接文本（如 “登录” 或 “注册” 中包含 “登”）

By.cssSelector() - CSS 选择器

CSDN-abments-Selenium CSS 选择器详细讲解

By.xpath() - XPath 表达式

WebDriver

WebDriver 是 Selenium 的核心接口，用于控制浏览器的行为。

控制函数

浏览器生命周期控制
启动与关闭

webdriver.Chrome() - 启动 Chrome 浏览器

webdriver.Firefox() - 启动 Firefox 浏览器

webdriver.Edge() - 启动 Edge 浏览器

webdriver.Safari() - 启动 Safari 浏览器

webdriver.Ie() - 启动 Internet Explorer

driver.quit() - 完全退出浏览器和WebDriver

driver.close() - 关闭当前窗口

会话管理

driver.session_id - 获取当前会话ID

driver.capabilities - 获取浏览器能力信息

driver.command_executor - 访问底层命令执行器

导航控制

页面导航

driver.get(url) - 加载指定URL

driver.back() - 浏览器后退

driver.forward() - 浏览器前进

driver.refresh() - 刷新当前页面

URL信息

driver.current_url - 获取当前URL

driver.title - 获取页面标题

driver.page_source - 获取页面HTML源码

窗口和框架控制

窗口管理

driver.current_window_handle - 当前窗口句柄

driver.window_handles - 所有窗口句柄列表

driver.switch_to.window(handle) - 切换窗口

driver.switch_to.new_window(type) - 打开新窗口/标签页

driver.maximize_window() - 最大化窗口

driver.minimize_window() - 最小化窗口

driver.fullscreen_window() - 全屏窗口

driver.set_window_size(width, height) - 设置窗口尺寸

driver.set_window_position(x, y) - 设置窗口位置

driver.get_window_size() - 获取窗口尺寸

driver.get_window_position() - 获取窗口位置

driver.get_window_rect() - 获取窗口位置和尺寸

框架管理

driver.switch_to.frame(index/name/id/webelement) - 切换frame

driver.switch_to.parent_frame() - 切换到父frame

driver.switch_to.default_content() - 返回主文档

元素操作

基本操作

element.click() - 点击元素

element.send_keys(“text”) - 输入文本

element.clear() - 清除输入

element.submit() - 提交表单

属性获取

element.text - 获取可见文本

element.get_attribute(“attr”) - 获取属性值

element.get_property(“prop”) - 获取DOM属性

element.value_of_css_property(“prop”) - 获取CSS属性

element.tag_name - 获取标签名

element.is_displayed() - 是否可见

element.is_enabled() - 是否可用

element.is_selected() - 是否被选中

element.size - 获取尺寸

element.location - 获取位置

element.rect - 获取位置和尺寸

element.screenshot(“filename.png”) - 元素截图

等待机制

Selenium 自动化 —— 四种等待（wait）机制

JavaScript 执行

driver.execute_script(script, *args) - 同步执行JS

driver.execute_async_script(script, *args) - 异步执行JS

script：要执行的 JavaScript 代码字符串

*args：可选参数，可以传递给 JavaScript 的参数

弹窗处理

driver.switch_to.alert - 获取alert对象

alert.text - 获取文本

alert.accept() - 确认

alert.dismiss() - 取消

alert.send_keys(“text”) - 输入文本

Cookies 管理

driver.get_cookies() - 获取所有cookies

driver.get_cookie(name) - 获取指定cookie

driver.add_cookie(cookie_dict) - 添加cookie

driver.delete_cookie(name) - 删除指定cookie

driver.delete_all_cookies() - 删除所有cookies

文件上传

element.send_keys(“/path/to/file”) (用于input[type=file]元素)

截图功能

driver.save_screenshot(“filename.png”) - 页面截图

driver.get_screenshot_as_base64() - 获取base64编码截图

driver.get_screenshot_as_png() - 获取二进制截图数据

日志管理

driver.get_log(log_type) - 获取浏览器日志

driver.log_types - 可用日志类型

常见问题与解决方法

WebDriver 版本不匹配

确保 WebDriver 的版本与浏览器的版本一致。

元素定位失败

使用浏览器开发者工具（F12）检查页面元素的属性，选择合适的定位方式。

脚本执行速度过快

在关键操作之间添加适当的等待时间或使用显式等待。
